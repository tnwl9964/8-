리스트

홈페이지에서 구조와 형태가 비슷한 연관되어있는 컨텐츠를 나타낼 때 리스트태그를 사용한다.

리스트는 크게 3가지 형태가 있다. 순서형 리스트/비순서형리스트/정의형리스트

순서형리스트

리스트들 사이에 순서에 흐름이 있는 경우 사용하는 태그

order list 순서가 있는 리스트들의 모임

순서형리스트의 예시) 네이버실검10위

  1. 물을 끓인다
  2. 스프를 넣는다
  3. 면을 넣는다

비순서형리스트

리스트들 사이에 순서에 흐름이 없는경우 사용하는 태그

unordered list 순서가 없는 리스트들의 모임

  • 시베리안 허스키
  • 말라뮤트
  • 사모예드

정의형리스트 dl

정의할 대상에 대한 설명 맥락을 만들 때 사용한다.

dl태그는 특정한 큰 주제에 대한 여러가지 정의항목을 만들 때 사용한다.

예시)음식의 레시피정보/ 상품의 세부정보 / 탭메뉴 컨텐츠

정의할대상1
정의할대상1에 대한 설명1
정의할대상1에 대한 설명2
정의할대상2
정의할대상2에 다한 설명
