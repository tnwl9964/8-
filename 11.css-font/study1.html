font

  • font-size -> 글씨의 크기를 지정하는 속성

    브라우저(html태그)의 글씨크기 기본값은 16픽셀이다.

    글씨의 가장 높은 부분에서 낮은 부분까지의 갈이를 기준으로 한다.(글씨체의 따라 다르게 적용될 수 있다)

    1em -> 부모태그에 적용된 글씨크기의 1배만큼

    1rem -> html태그에 적용된 글씨크기의 1배만큼

  • line-height -> 행간(글씨를 감싸는 보이지않는 라인의 높이)

    행간은 기본적으로 글씨크기의 120%만큼 설정되어 있다.

    120%는 소수로 1.2라고 표현해도 된다. 글씨크기가 50픽셀만큼 설정되어있을 때 line-height가 3만큼 지정되어있다면 계산된 행간은 150픽셀이 된다. 글씨크기를 제외한 100픽셀을 위아래로 똑같이 나누어 여백을 만들게 된다.

    • lighter - 얇게
