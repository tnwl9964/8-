alert()

알림창을 띄우는 함수 - 괄호안에 알림창에 보여질 데이터를 입력.

console.log()

콘솔창에 로그를 띄우는 함수 - 괄호안에는 콘솔창에 보여질 데이터를 입력.

document.write()

body태그에 컨텐츠를 필기하는 함수 - 괄호안에는 필기 할 데이터를 입력.

Number()

괄호안에 숫자처럼 생긴 문자데이터를 숫자로 변형하여 함수가 사용된 자리에 변형된 결과값을 남기는 함수.

typeof()

괄호안에 데이터의 형태를 검사하고 검사한 결과값(문자데이터형태로)을 함수가 사용된 자리에 남기는 함수.

prompt()

홈페이지를 사용자에게 괄호안에 질문을 한 후 사용자가 입력한 대답을 함수가 사용되었던 자리에 문자데이터 형태로 남게주는 함수.

수학계산관련 함수

모두 결과를 게산해서 함수가 사용되었던 자리에 결과 값을 남긴다.
