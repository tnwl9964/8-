태그 선택

$("css선택자") -> $는 jQuery함수라는 뜻으로 문서 안에서 css선택자 조건을 갖는 태그를 선택하여 함수가 사용된 자리에 제이쿼리 문서객체형태로 리턴한다.

css메서드

css메서드는 제이쿼리 문서객체에 사용되는 스타일을 바꿔주는 메서드이다.

get메서드

get()메서드를 통하여 제이쿼리 문서객체를 순수자바스크립트 문서객체로 잔환할 수 있다.

ready메서드

이 함수는 문서가 body태그 끝까지 읽혀진 뒤 한번 실행되는 예약 함수로 함수의 매개변수로는 동작을 넣어 표현한다.

window.onlond메서드와 비슷하다.

attr메서드

attr함수를 통해서 태그의 속성값을 변경할 수 있다.(매게변수 2개) / 속성에 적용된 속성값을 불러올 수 있다.(매게변수 1개)

네이버로 바로가기

text/html메서드

선택한 태그의 내용을 바꿀 떄 사용하는 함수이고 태그의 내용에 태그문법이 포함된경우는 html메서드를 활용하고 그렇지 않으면 text메서드를 사용한다.

선택한 태그의 내용을 가져오려고 할때는 text()를 매게변수를 입력하지 않고 사용하면 가능하다.

임의의 내용

포함관계를 이용한 태그 선택

  • next() - 선택한 태그 기준 다음형제 태그
  • prev() - 이전형태의 태그
  • children() - 자식태그선택
  • parent() - 부모태그선택

eq메서드

선택한 태그가 여러개인 경우 eq()함수를 사용하여 순번을 입력하여 선택할 수 있다.

순번은 0부터 카운트

  • list1
  • list2
  • list3
  • list4
  • list5
  • list6
  • list7

클래스 함수

선택한 태그의 클래스를 추가하거나 삭제할 수 있다.
