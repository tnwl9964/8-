레이아웃(배치)방법 - flex

블록레벨태그들을 좌우로 나란히 배치하는 방법으로 flex방법을 배워보자.

플렉박스에는 크게 부모태그(플렉스컨테이너) 자식태그(플렉스아이템) 두 부분으로 나뉘며 각각의 부분마다 사용할 수 있는 css속성이 다르다.

플렉스아이템들은 플렉스컨테이너의 자식테그들을 말하며, 기본적으로는 플렉스컨테이너 안에서 컨테이너의 메인축을 따리서 배치되는 특징을 갖는다.

교차축은 메인축의 방향에 따리서 자동으로 바뀐다.

flex-container(대장)에 지정하는 속성

  • display:flex -> 이 속성이 적용된 태그가 플렉스컨테이너(대장)가 된다.

    flex-container가 된 태그는 좌에서 우 방향으로(row방향)으로 메인축이 형성되고, 위에서 아래방향으로(column방향) 교차축이 형성된다.

    메인축과 교차축의 시작점과 마지막점은 각각 start지점 end지점이라 부른다.

    컨태이너의 자식들인 아이템들은 무조건 메인축을 따라서 나란히 배치가 된다.

  • flex-direction -> 메인축의 방향을 바꾸어 주는 속성

    • row(기본값) - 좌->우
    • row-reverse - 우->좌
    • column - 위->아래
    • column-revers -> 아래->위

    플렉스아이템들은 무조건 메인축의 시작점부터 나란히 배치된다.

    • justify-content -> 메인축을 따라 아이템을 정렬하는 방법
    • flex-start(기본값) - 아이템들은 메인축의 시작점에서 정렬
    • flex-end - 메인축의 마지막지정에 정렬
    • center - 메인축 중앙지점에 아이템 정렬
    • space-between - 메인축의 시작과 마지막지점에 아이템을 맞춘 후 아이템들의 사이여백을 동일하게 지정하도록 정렬
    • space-around - 각 아이템들을 좌우 여백을 동일하게 지정하여 정렬

  • align-items -> 교차축을 따라 아이템을 정렬하는 방법

    • stretch(기본값) - 교차축 길이만큼 아이템길이를 늘림
    • flex-start - 교차축의 시작지점에 아이템 정렬
    • flex-end - 교차축의 마지막지점에 아이템 정렬
    • center - 교차축의 중앙지점에 아이템 정렬

  • flex-wrap - flex-item들의 줄바꿈을 설정하는 속성

    가본값은 nomwrap 줄바꿈을 허용하지 않는다.

    wrap 아이템들의 줄바꿈을 허용

  • aling-content

    flex-container에서 아이템들의 줄바꿈을 허용했을 때 container전체 높이를 수직축으로하여 아이템그룹 전체를 정렬할 때 사용하는 속성.
