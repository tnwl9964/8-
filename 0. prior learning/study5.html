boolean데이터

값이 두개 true / false 밖에 없다.

비교

어떤 명확한 비교를 통해서 참인지 거짓인지 계산된다.

예)나는 빵이2개가 있고 너는 빵이 1개 있다 그럼 내가 너보다 빵이 많다는 것은 사실이다. -> 2>1

!은 부정의 의미로 사용된다.

사실에 대한 존재유무

~이 있다면 true / ~이 없다면 false

false로 간주되는 경우

  • 숫자데이터인데 값이 0인 경우
  • 문자데이터인데 빈문자데이터인 경우
  • 변수인데 값이 저장되지 않는 빈 변수
