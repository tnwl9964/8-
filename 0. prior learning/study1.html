태그의 속성

태그의 부가적인 정보를 입력할 때 사용하는 문법

속성은 시작태그에 한칸띄어서 구분하여 입력한다.

속성명="속성값" 형식으로 입력한다

속성은 모든 태그애서 모두 사용가능한 글로벌속성과 특정태그에서만 사용할 수 있는 로컬속성 두가지가 있다.

태그에 속성은 2개이상 여러개를 입력할 수 있으며 순서는 관계없다.

링크태그

href속성에 적용된 경로로 요청을

경로의 종류

절대경로

웹 상에서 긁어서 가져온 경로는 전부 절대경로다. (어디서든 동일하게 작동하는 경로)

상대경로

현재파일에서 목적지 파일까지의 경로

. -> 현재 파일의 위치를 불러온다
..-> 현재 파일의 기준 상위폴더의 위치를 불러온다
/ -> 폴더를 연다.
