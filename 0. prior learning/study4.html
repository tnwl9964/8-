표

table 태그를 만들면 홈페이지에 보이지않는 표공간이 만들어진다.

표안에서 행을 만들 떄 사용하는 태그는 tr태그이다.

행안에 셀들을 입력할 떄 사용하는 태그는 td/th태그이다.

제목셀(th) / 일반내용셀(td) 사용한다.
