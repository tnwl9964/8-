높이가 없으면 즉 태그공간이 없으면 배경색도 포함이 안 된다.

background

컨텐츠가 아닌 배경 혹은 배경이미지를 넣을 때 사용하는 속성

태그의 공간(테두리 안쪽 모든 공간)에 배경색 또는 배경이미지를 나타낼 때 사용하는 css속성

background-color

배경색상 입력

background-image

배경이미지 입력 - url()도구 사용

배경이미지는 입력하면 자동으로 가로세로로 반복된다.

background-repeat

배경이미지가 반복되는 속성

repeat(기본값-가로로세로로 반복)

no-repeat 반복안함

repeat-x 가로로만 반복

repeat-y 세로로만 반복

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background:x좌표 y좌표

키워드방식, %방식, 픽셀방식

background-size

배경이미지의 가로세로사이즈를 지정하는 속성

축약형 표현

색상 이미지 반복속성 좌표 / 사이즈 순으로 작성한다.

background-attachment

기본값 scroll - 마우스스크롤을 내릴 때 배경이미지도 같이 스크롤에 따라 올라감

fixed - 마우스스크롤을 내릴 때 배경이미지는 화면에 고정

background-blend-mode

배경색과 배경이미지를 혼합할 때 사용하는 속성
