가상클래스(추상클래스)

사용자의 행동양식을 선택자로 표현할 수 있다.

  • :hover - 사용자가 마우스를 올렸을 때의 스타일을 미리 지정할 수 있다.
  • :active - 마우스를 클릭했을 때(클릭을 한 뒤에 마우스를 떼지 않은 상태)
  • a:link - 방문한 적이 없는 링크
  • a:visited - 방문한 적이 있는 링크
  • :before - 선택한태그의 첫번째 자식태그를 생성
  • :after - 선택한태그의 마지막번째 자식태그를 생성

가상태그 before와 after태그는 css에서 일괄적으로 선택한 태그들에게 척번째나 마지막 자식태그를 만들 때 사용하는 태그이다.

반드시 컨택트라는 css속성이 명시되어있어야 before와 after태그가 만들어진다.
